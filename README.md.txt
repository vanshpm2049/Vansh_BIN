# 🏙️ Enhanced Smart Waste Management Dashboard

An interactive **Streamlit Dashboard** for analyzing and visualizing waste management data with **3D visualizations, AI-powered predictions, and geographic mapping**.

🚀 Built with **Streamlit, Plotly, PyDeck, Folium, and Scikit-Learn**.

---

## 📌 Features
- 🎮 **3D Visualizations**: Rectangular bars, hexagon aggregation, cylindrical columns, scatter bubbles  
- 🗺️ **Geographic Maps**: Interactive city/community mapping with Folium  
- 🤖 **AI Insights**: Machine learning waste prediction models  
- 📊 **Analytics**: Metrics, trends, correlations, and efficiency analysis  
- 📥 **Flexible Data Input**: Use built-in GIS sample data or upload your CSV
